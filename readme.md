# My to-do web app

I made this to-do app to learn the basics of HTML/CSS/JavaScript.

## Features

* Add list entries
* Mark list entry as completed
* Clear completed entries from list
* Edit list entries
* Save the list in storage [Planned]
* Create multiple lists [Planned]
